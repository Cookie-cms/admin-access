SQL Guide

Этот гайд поможет вам понять основы работы с SQL — языком запросов для работы с реляционными базами данных.

1. SELECT — Выборка данных

Команда SELECT используется для извлечения данных из таблиц.

SELECT * FROM users;

Этот запрос извлекает все строки и столбцы из таблицы users.

Вы также можете ограничить результаты, используя условие WHERE:

SELECT username, mail FROM users WHERE mail_verify = 1;

2. INSERT INTO — Добавление данных

Команда INSERT INTO используется для добавления новых записей в таблицу.

INSERT INTO users (username, mail) VALUES ('john_doe', 'john@example.com');

Этот запрос добавит новую строку в таблицу users с указанными значениями для столбцов username и mail.

3. UPDATE — Обновление данных

Команда UPDATE используется для изменения существующих записей в таблице.

UPDATE users SET mail = 'new_email@example.com' WHERE id = 1;

Этот запрос обновляет столбец mail в таблице users, где id равно 1.

4. DELETE — Удаление данных

Команда DELETE используется для удаления данных из таблицы.

DELETE FROM users WHERE id = 1;

Этот запрос удаляет запись из таблицы users, где id равно 1.

5. ALTER TABLE — Изменение структуры таблицы

Команда ALTER TABLE используется для изменения структуры таблицы, например, добавления, изменения или удаления столбцов.

ALTER TABLE users ADD COLUMN age INT;

Этот запрос добавляет новый столбец age в таблицу users.

6. JOIN — Объединение таблиц

Запрос JOIN используется для объединения данных из нескольких таблиц.

SELECT users.username, orders.amount FROM users JOIN orders ON users.id = orders.user_id;

Этот запрос объединяет таблицы users и orders, выбирая имена пользователей и их заказы.

7. GROUP BY — Группировка данных

Запрос GROUP BY используется для группировки данных по одному или нескольким столбцам.

SELECT mail_verify, COUNT(*) FROM users GROUP BY mail_verify;

Этот запрос группирует пользователей по столбцу mail_verify и подсчитывает количество пользователей в каждой группе.

8. ORDER BY — Сортировка данных

Запрос ORDER BY используется для сортировки данных по указанному столбцу.

SELECT * FROM users ORDER BY username ASC;

Этот запрос сортирует пользователей по столбцу username в порядке возрастания.

9. LIMIT — Ограничение количества строк

Запрос LIMIT используется для ограничения количества возвращаемых строк.

SELECT * FROM users LIMIT 5;

Этот запрос вернет только первые 5 строк из таблицы users.

10. Основные операторы

  • AND — логическое И (для комбинирования условий): 
    SELECT * FROM users WHERE mail_verify = 1 AND perms = 1;
  • OR — логическое ИЛИ (для комбинирования условий): 
    SELECT * FROM users WHERE mail_verify = 1 OR perms = 1;
  • LIKE — поиск по шаблону: 
    SELECT * FROM users WHERE username LIKE 'john%';
  • IN — проверка на наличие значения в списке: 
    SELECT * FROM users WHERE id IN (1, 2, 3);
  • BETWEEN — поиск значений в диапазоне: 
    SELECT * FROM users WHERE id BETWEEN 1 AND 10;
  • IS NULL — проверка на NULL: 
    SELECT * FROM users WHERE username IS NULL;
